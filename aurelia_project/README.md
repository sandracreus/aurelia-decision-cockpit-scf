# Aurelia Decision Cockpit (Streamlit)

App educativa con:
1) Representación S/A/T/G/C/R
2) Predictor ML simulado
3) Optimizador OR (greedy knapsack)
4) Sistema híbrido ML+OR
5) Copiloto LLM (auditoría simulada)
 
## Ejecutar (en Streamlit Cloud)
Entrypoint: `aurelia_project/app.py`
 
## Estructura
- `aurelia_project/app.py`: UI + navegación
- `aurelia_project/modules/*`: lógica por módulo
- `cases/catalog.yml`: plantillas de casos
